Ladybird Anti-Fingerprinting Engine
C++ · 10,000+ lines
Platform-coherent hardware identity generation system built into a custom Ladybird browser fork. Generates internally consistent device profiles enforced at the C++ component level — not a JS patch layer. Platform constraints are hard-enforced: Windows profiles get ANGLE/D3D GPUs, macOS gets Apple Silicon/Metal, iOS gets Apple SoC with no discrete GPU, Linux gets Mesa DRI. Cross-contamination is blocked in code. Covers audio (oscillator noise + FFT bin offsets), canvas, WebGL variants and constraints, navigator, geolocation, high-resolution timing, element timing, resource timing, workers, clipboard, compression, cookies, form input, navigation, UI events, and GPC — the entire fingerprinting surface.
