Ladybird Anti-Fingerprinting Engine
A privacy-focused browser built on the Ladybird open-source engine with 10,000+ lines of custom C++. The core anti-fingerprinting system generates complete, internally consistent hardware and software device identities enforced directly at the C++ component level — not a JavaScript patch layer, which is trivially detectable by any modern fingerprinting script.

Platform constraints are hard-enforced: Windows profiles use ANGLE/D3D GPUs, macOS uses Apple Silicon/Metal, iOS uses Apple SoC with no discrete GPU, Linux uses Mesa DRI. Cross-platform contamination is explicitly blocked. Covers the full fingerprinting surface: audio, canvas, WebGL, navigator, geolocation, high-resolution timing, workers, UI events, clipboard, and GPC.
