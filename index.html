02 Projects
C++ · Browser Engineering · Privacy
Ladybird Anti-Fingerprinting Engine
Platform-coherent hardware identity generation at C++ component level

Built a privacy-focused browser on top of the Ladybird open-source engine with 10,000+ lines of custom C++. Developed a core anti-fingerprinting system that generates complete, internally consistent hardware and software device identities enforced directly at the C++ component level — not a JavaScript patch layer, which is trivially detectable. Platform constraints are hard-enforced: Windows profiles use ANGLE/D3D GPUs, macOS uses Apple Silicon/Metal, iOS uses Apple SoC with no discrete GPU, Linux uses Mesa DRI. Cross-platform contamination explicitly blocked. Covers the full fingerprinting surface: audio, canvas, WebGL, navigator, geolocation, high-resolution timing, workers, UI events, clipboard, and GPC.
